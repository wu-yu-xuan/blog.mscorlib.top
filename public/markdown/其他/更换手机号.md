# 更换手机号

## 目的/槽点

1. 对本次更换手机号做一个 check list
2. 为下次更换手机号列个 todo list
3. 一些再也不用的软件不参与此次更换手机号，如微博、百度、知乎
4. 看看这傻逼制度有多恶心，换手机号要求发短信至上个手机号的都是傻逼
5. 手机更换 SIM 卡时要重启，不然收不到短信
